4

Write and test an interface : IVehicle
IVehicle ensures the following methods:
  • int GetVehicleId()
  • string GetName()
  • string GetVehicleType()
Write the classes Bus and Car and let those two classes implement the interface.
The constructor fills an int: vehicleId, a string: name and a string: vehicleType.
Make several objects of the classes. Check if the interface is implemented.

5

An old - and not especially advanced - way to decide how difficult a programming task is is to calculate Function Points:
The idea was to calculate the number of lines with a level of difficulty
(example: Factors: 3,6,9) can be used for calculation.
Example:
10 lines of code with factor 3 = 30 p
15 lines of code with factor 6 = 90 p
Write and test a simple class that implement IComparable for
listning of the work (starting with greatest FP).
The class ( name for instance FP ) contains the variables:
int numberOfLines
int factor
Make several objects ( with different FP's ) and sort them.
