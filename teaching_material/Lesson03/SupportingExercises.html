Supporting exercises - Lesson 3

1

Write and test an application with a class which inherits the Complex class from Lesson 2. The class could be a PolarForm class extending the Complex class.

2

Write and test a class which uses simple file manipulation ( see chp. 20 for inspiration).
Try to make this class as general as possible. Now the class can be inherited from another class which specialize the file manipulation class.

3

A singleton (GoF) pattern ensures that only one instance of a class can be active at a time. Try to create a class supporting this pattern. Hint: Use a static method and a private constructor.

4

Reuse the inventory application from lesson 2: Create some specialized classes that inherits from Article. Try to use the "is" and "as" keywords to place specialized articles on a separate list.

5

(Polymorphy) Team Danmark support many sport types. A sport can be team sport or an individual sport. Make a base class Sport with the following attributtes:
  • string sportType
  • double amount
With the following methods:
  • A constructor
  • PrintData
The classes HandBall, TableTennis (maybe others) inherit from Sport and has as a minimum the method PrintData.

6

Try new features from C#9.
