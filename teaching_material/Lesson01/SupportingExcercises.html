7 (extra)

Write your own Intermediate Language:
Here follows a small example:
.assembly
extern mscorlib{}.assembly
Hejsa
{
.ver 1:0:1:0
}
.module Hejsa.exe
.method static void Main() cil managed
{
.maxstack 1
.entrypoint
ldstr "Hallo from IL"
call void [mscorlib]System.Console::WriteLine(string)
ret
}

Use ilasm ( Intermediate Assembler) for assembling.
